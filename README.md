# Dot-com Bubble vs AI Market Boom

## Overview

This project explores the similarities and differences between the Dot-com Bubble (1995–2002) and the recent AI-driven market expansion (2022–Present).

Using historical stock market data and financial metrics, the project compares price behavior, volatility, valuation, market concentration, and drawdowns to investigate whether the current AI boom exhibits characteristics similar to those observed during the Dot-com era.

---

## Objectives

- Compare stock price performance across both periods
- Analyze market volatility
- Measure drawdowns and recoveries
- Compare valuation metrics
- Investigate market concentration
- Build reproducible data pipelines using Python
